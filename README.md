# rails5-docker

```bash
docker-compose run web rails new . --force --database=postgresql --skip-bundle
docker-compose build
```

Edit ```config/database.yml```
```yaml
development: &default
  adapter: postgresql
  encoding: unicode
  database: postgres
  pool: 5
  username: postgres
  password:
  host: db

test:
  <<: *default
  database: myapp_test
  ```
  ```bash
  docker-compose up
  docker-compose run web rake db:create
  ```
